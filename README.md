# Elm single-file starter

This template builds an Elm app and inlines both the compiled Elm bundle and the Tailwind CSS generated by Vite into one final HTML file.

## Build

```bash
npm install
npm run build
```

The output lands in `dist/index.html`. This template expects the dependencies to be installed locally so Vite can load the Tailwind plugin cleanly.

## Edit points

- `src/Main.elm` for the Elm app
- `src/styles.css` for Tailwind layers and base styles
- `src/vite-entry.js` for the Vite CSS entry
- `vite.config.mjs` for the Tailwind/Vite pipeline
- `src/index.template.html` for the final HTML shell
